### Модуль Warden Features для Jailbreak Core  
(https://github.com/your-repo/jailbreak-core) — добавляет настройки для Начальника Тюрьмы при его назначении: изменение здоровья, брони, скорости, цвета модели и проигрывание звуков.

При назначении нового Начальника Тюрьмы автоматически применяются параметры из конфигурационного файла `addons/configs/Jailbreak/warden_features.ini`.

Также модуль поддерживает:
- установку здоровья и брони
- изменение скорости передвижения
- смену цвета модели
- проигрывание глобального и локального звука при назначении

### Установка:
Распаковать в `game/csgo/addons/`

### Требования:
- [Jailbreak Core](https://discord.gg/WkTwuKe8zy)
- [Utils](https://github.com/Pisex/cs2-menus)
